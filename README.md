# todomvc
This project is a todomvc demo.

## Technology
- Backend: `Go`, `Beego`
- Frontend: `React`, `Redux`

## Installation
1. Build frontend
```
cd $GOPATH/src/todomvc/views
npm install
npm run build
```
2. Run application
```
cd $GOPATH/src/todomvc
bee run
```

## Usage
```
http://127.0.0.1:8080
```
